PulseQuill


PulseQuill is my portfolio for my 2nd year of College as a Computer Science student at the University of Perpetual Help System DALTA in
Molino Campus. This contains all of my activities and assessments for Design, Analysis, and Algorithm Technologies in laboratory
and lecture. Feel free to explore the complexity, yet amazing algorithms I have implemented. You may contact me through my socials below.


Activity 1

Counting Algorithm


The program counts males, females, and computers in a computer lab using a for loop, converting uppercase entries to lowercase for accuracy, and printing final counts.

Activity 2

Even Number Filter


The program calculates the first digit of a csv file by looping through the "Number1" column, extracting the first digit, and printing the result.

Activity 3

Comb Sort Algorithm


The program uses the comb sort algorithm to access the csv file dataset and prompts the user to sort the "No of ratings" column in ascending or descending order

Activity 4

Ford-Fulkerson Algorithm


The program uses the Ford-Fulkerson Algorithm to solve a flow problem for assigning study hours based on student data, integrating data processing, visualization, and CSV export.

Activity 5

Chatbot


A friendly trained Ollama chatbot that is knowledgeable in linear, and binary search algorithms, regarding its history, sample codes, application, and such.
